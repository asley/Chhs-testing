# Gibbon Platform Summary

Gibbon is a free, open-source school management platform. It provides a flexible and comprehensive solution for schools to manage their data and administrative tasks.

## Technology Stack

*   **Backend:** PHP, with MySQL as the database.
*   **Frontend:** HTMX and Alpine.js (from version 28 onwards), replacing older libraries like jQuery.

This document confirms the initial setup and familiarization with the Gibbon project.
